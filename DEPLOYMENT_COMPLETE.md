
**Backend URL:** `http://localhost:12001`  
**Health Check:** `http://localhost:12001/api/health`

### Frontend Deployment

```bash
# Navigate to frontend
cd SalesSync/frontend-vite

# Install dependencies
npm install

# Build for production
npm run build

# Serve (using serve or nginx)
npm run preview
# OR
serve -s dist -l 12002
```

**Frontend URL:** `http://localhost:12002`

### SSL Deployment

SSL Certificate provided: `/workspace/project/SSLS.pem`

For HTTPS deployment:
```bash
# Use with nginx
server {
    listen 443 ssl;
    ssl_certificate /path/to/SSLS.pem;
    ssl_certificate_key /path/to/SSLS.key;
    
    location / {
        proxy_pass http://localhost:12002;
    }
    
    location /api {
        proxy_pass http://localhost:12001;
    }
}
```
